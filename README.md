# osuDirectMirror
一个小工具，用于修改 osu!Direct 使用的 Mirror，也许这将使部分玩家的下载速度提高。  
请注意: 只有 osu!Supporter 才可以使用 osu!Direct，本工具可能不适用于第三方 osu! 服务器。
